## 页面描述

观察模式结束后展示的分析报告页面，包含课程摘要、量化指标统计、学生个体对比等内容。用户可在此查看完整的教学效果分析。

## 组件列表

### 1. 导航栏
- **位置**：页面顶部
- **内容**：
  - 左侧：返回按钮（← 返回）
  - 中间：页面标题"观察模式 - 分析报告"
- **样式**：
  - 背景：白色半透明
  - 底部边框：2px 手绘虚线

### 2. 课程摘要卡片
- **样式**：
  - 背景：蓝色便签 `var(--sticky-blue)`
  - 边框：2px 马克笔蓝色
  - 内边距：20px
- **内容**：
  - **第一行**：
    - 教学主题："Python变量与数据类型"（24px 粗体）
    - 分隔符："|"
    - 教学模式标签："启发式"（紫色标签）
    - 分隔符："|"
    - 学生数量："10名学生"
  - **第二行**：
    - 开始时间："开始时间: 10:00"
    - 分隔符："|"
    - 结束时间："结束时间: 11:15"
    - 分隔符："|"
    - 总时长："总时长: 75分钟"（高亮显示）

### 3. 量化指标网格
- **布局**：
  - 标题："量化指标"（20px 粗体）
  - 网格：3列 × 2行，间距 16px
- **指标卡片**：
  - 每个卡片：160px × 120px
  - 背景：白色
  - 边框：2px 手绘边框
  - 阴影：3px 3px 0px rgba(0,0,0,0.1)

#### 指标列表
1. **总消息数**：156
   - 图标：💬
   - 数值：48px 粗体
   - 单位：条

2. **互动频率**：2.1/分钟
   - 图标：⚡
   - 数值：48px 粗体
   - 说明：每分钟平均消息数

3. **学生参与率**：85%
   - 图标：🤚
   - 数值：48px 粗体
   - 进度条：显示百分比

4. **平均知识掌握度**：72.5%
   - 图标：🧠
   - 数值：48px 粗体
   - 进度条：显示百分比
   - 颜色：绿色（>70%）

5. **平均正确率**：68.3%
   - 图标：✅
   - 数值：48px 粗体
   - 进度条：显示百分比
   - 颜色：黄色（60-80%）

6. **检查点完成率**：100% (5/5)
   - 图标：🎯
   - 数值：48px 粗体
   - 说明：已完成检查点数

### 4. 学生个体统计表格
- **样式**：
  - 背景：白色卡片
  - 边框：2px 手绘边框
  - 内边距：20px
- **表格头部**：
  - 列：姓名 | 等级 | 发言次数 | 正确率 | 知识掌握度 | 态度
  - 背景：浅灰 `var(--bg-tertiary)`
  - 文字：14px 粗体

#### 表格行
- **学生行**：
  - 背景：白色 / 斑马纹（奇数行浅灰）
  - 边框：底部 1px 虚线
  - 悬停效果：背景变为浅蓝

- **列内容**：
  1. **姓名**：学生姓名 + 头像（首字）
  2. **等级**：A/B/C 标签
     - A：绿色标签
     - B：黄色标签
     - C：红色标签
  3. **发言次数**：数字
  4. **正确率**：百分比 + 颜色编码
     - >80%：绿色
     - 50-80%：黄色
     - <50%：红色
  5. **知识掌握度**：百分比 + 进度条
  6. **态度**：积极/中立/被动
     - 积极：绿色标签
     - 中立：黄色标签
     - 被动：红色标签

#### 排序功能
- 点击列头可按该列排序
- 排序指示器：↑↓ 箭头
- 默认按知识掌握度降序排列

### 5. 底部操作区
- **布局**：居中对齐
- **按钮**：
  1. **导出报告**：主按钮，蓝色
     - 图标：📥
     - 文字："导出报告"
  2. **返回首页**：次要按钮
     - 图标：🏠
     - 文字："返回首页"

## 交互说明

### 页面加载
- 从后端获取会话报告数据
- 显示加载动画
- 数据加载完成后渲染页面

### 表格排序
- 点击列头触发排序
- 支持升序/降序切换
- 排序后更新表格内容

### 导出报告
- 点击"导出报告"按钮
- 生成 PDF 或 Excel 格式报告
- 自动下载到本地
- 显示成功提示

### 返回首页
- 点击"返回首页"按钮
- 跳转到首页 `/`

### 返回按钮
- 点击导航栏返回按钮
- 返回到观察界面查看消息回放

## API 接口

```typescript
// 获取会话报告
GET /observation/{session_id}/report

Response:
{
  "summary": {...},
  "metrics": {...},
  "students": [...]
}

// 导出报告
POST /observation/{session_id}/export

Request:
{
  "format": "pdf" | "excel"
}

Response: File download
```

## 设计备注

- 量化指标使用大数字突出显示
- 进度条和颜色编码使数据更直观
- 表格支持排序，便于对比学生表现
- 导出功能方便保存和分享报告
- 保持整体视觉层次清晰
- 所有数据以可视化方式呈现，减少阅读负担
