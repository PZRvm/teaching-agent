## 页面描述

教学历史会话列表页，展示所有历史教学会话。支持搜索、筛选功能，可按关键词搜索主题、按教学模式筛选、按模式类型筛选。点击卡片跳转到会话详情页。

## 组件列表

### 1. 导航栏
- **位置**：页面顶部
- **内容**：
  - 左侧：返回按钮（← 返回）
  - 中间：页面标题"教学历史"
- **样式**：
  - 背景：白色半透明
  - 底部边框：2px 手绘虚线

### 2. 搜索与筛选区
- **样式**：
  - 背景：白色卡片
  - 边框：2px 手绘边框
  - 内边距：20px
- **组件**：

#### 搜索框
- **样式**：
  - 宽度：100%
  - 高度：48px
  - 手绘边框输入框
  - 左侧搜索图标：🔍
- **属性**：
  - 占位符："搜索主题、教师、学生..."
  - 支持实时搜索（输入后300ms触发）

#### 筛选下拉框
- **布局**：横向排列，间距 16px
- **下拉框**：

1. **教学模式筛选**
   - 标签："教学模式："
   - 下拉框：[全部 ▼]
   - 选项：
     - 全部
     - 灌输式
     - 启发式
     - 讨论式

2. **模式类型筛选**
   - 标签："模式类型："
   - 下拉框：[全部 ▼]
   - 选项：
     - 全部
     - 观察模式
     - 教师模式

### 3. 会话列表
- **样式**：
  - 背景：白色卡片
  - 边框：2px 手绘边框
  - 内边距：20px
- **布局**：垂直排列，间距 16px

#### 会话卡片
每个会话显示为一个卡片，包含：

- **样式**：
  - 背景：黄色便签 `var(--sticky-yellow)`
  - 边框：2px 手绘边框
  - 内边距：16px
  - 悬停效果：阴影加深，轻微上移

- **内容**：
  1. **教学主题**（24px 粗体）
     - 例如："Python变量与数据类型"

  2. **标签行**（16px）
     - 模式类型标签："观察模式"（蓝色）或"教师模式"（绿色）
     - 教学模式标签："灌输式"（黄色）/ "启发式"（紫色）/ "讨论式"（粉色）
     - 学生数量："10名学生"
     - 会话日期："2026-04-08"

  3. **状态和时长行**（14px）
     - 状态标签：
       - "已结束"（灰色）
       - "进行中"（绿色，带动画）
     - 时长：
       - 已结束："时长：75分钟"
       - 进行中："已进行：45分钟"（实时更新）

  4. **操作按钮**
     - 已结束："[查看详情 →]"
     - 进行中："[继续观察 →]" 或 "[继续教学 →]"
     - 按钮样式：次要按钮，小尺寸

#### 空状态
- **触发条件**：没有匹配的会话记录
- **内容**：
  - 图标：📭（64px）
  - 标题："暂无教学记录"（20px 粗体）
  - 提示："开始第一次教学来创建记录吧"（16px）
  - 按钮："[开始观察]" / "[开始教学]"

### 4. 加载更多按钮
- **位置**：列表底部
- **样式**：次要按钮，全宽
- **内容**："加载更多..."
- **状态**：
  - 有更多数据：正常显示
  - 加载中：显示加载动画
  - 无更多数据：隐藏或禁用

## 交互说明

### 搜索功能
- 在搜索框输入关键词
- 实时搜索（300ms 防抖）
- 搜索范围：教学主题、学生姓名
- 匹配的会话实时显示在列表中
- 清空搜索框后恢复显示所有会话

### 筛选功能
- 点击教学模式下拉框选择教学模式
- 点击模式类型下拉框选择模式类型
- 选择后列表实时更新
- 支持"全部"选项（显示所有会话）
- 搜索和筛选可同时使用（AND 逻辑）

### 查看详情
- 点击"查看详情"按钮
- 跳转到会话详情页 `/history/detail/{session_id}`
- 已结束的会话显示完整报告

### 继续会话
- 进行中的会话显示"继续观察"或"继续教学"按钮
- 点击后跳转到对应页面：
  - 观察模式：`/observation/view?session_id={id}`
  - 教师模式：`/teacher/view?session_id={id}`

### 加载更多
- 滚动到底部或点击"加载更多"按钮
- 加载更多历史会话
- 分页加载（每页 20 条）
- 加载时显示加载动画

### 返回
- 点击导航栏的"返回"按钮
- 返回到首页 `/`

## API 接口

```typescript

GET /history/sessions?page=1&page_size=20&search={keyword}&teaching_mode={mode}&mode_type={type}

Response:
{
  "sessions": [...],
  "total": 50,
  "page": 1,
  "page_size": 20
}
```

## 设计备注

- 卡片式布局，每个会话一目了然
- 标签使用不同颜色区分模式类型和教学模式
- 进行中的会话使用绿色标签和实时时长，突出显示
- 搜索和筛选功能组合使用，方便快速定位
- 空状态提供友好提示和引导操作
- 加载更多按钮支持分页加载
- 保持整体视觉简洁清晰
- 悬停效果增强交互反馈
