# Local Fonts

Google Fonts downloaded for offline use in the rough-design skill.

## Fonts Included

### Be Vietnam Pro
- **Regular** (400): `BeVietnamPro-Regular.ttf`
- **Medium** (500): `BeVietnamPro-Medium.ttf`
- **SemiBold** (600): `BeVietnamPro-SemiBold.ttf`

### Plus Jakarta Sans
- **Regular** (400): `PlusJakartaSans-Regular.ttf`
- **SemiBold** (600): `PlusJakartaSans-SemiBold.ttf`
- **Bold** (700): `PlusJakartaSans-Bold.ttf`
- **ExtraBold** (800): `PlusJakartaSans-ExtraBold.ttf`

## Source

These fonts were downloaded from Google Fonts:
- https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;600;700;800&family=Be+Vietnam+Pro:wght@400;500;600&display=swap

## Usage

Import the fonts in your CSS:

```css
@import url('./fonts/fonts.css');
```

Or include in your HTML:

```html
<link href="./fonts/fonts.css" rel="stylesheet">
```

Then use the fonts:

```css
font-family: 'Plus Jakarta Sans', sans-serif;
font-family: 'Be Vietnam Pro', sans-serif;
```
