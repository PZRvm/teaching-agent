## 页面描述

检查点计划编辑器，在教学开始前展示系统自动生成的检查点计划。用户可以查看、编辑、新增、删除、调整检查点顺序，确认后开始正式教学。

## 组件列表

### 1. 导航栏
- **位置**：页面顶部
- **内容**：
  - 左侧：返回按钮（← 返回）
  - 中间：页面标题"检查点计划编辑器"
- **样式**：
  - 背景：白色半透明
  - 底部边框：2px 手绘虚线

### 2. 教学信息卡片
- **样式**：
  - 背景：绿色便签 `var(--sticky-green)`

- **内容**：
  - 教学主题："主题：Python函数的定义与调用"（20px 粗体）
  - 学生数量和预计时长："学生数：10人 | 预计时长：45分钟"（16px）

### 3. 检查点计划列表
- **样式**：
  - 背景：白色卡片
  - 边框：2px 手绘边框
  - 内边距：20px
  - 最小高度：500px
- **组件**：

#### 检查点卡片
每个检查点显示为一个卡片，包含：

- **样式**：
  - 背景：黄色便签 `var(--sticky-yellow)`
  - 边框：2px 手绘边框
  - 内边距：16px
  - 间距：16px

- **内容**：
  1. **检查点编号**："检查点 1"（16px 粗体，灰色）
  2. **标题**："Python函数简介"（24px 粗体）
  3. **核心知识点**："核心知识点：函数的概念、作用"（16px）
  4. **检查问题**："检查问题：函数是什么？"（16px，蓝色）

- **操作按钮**（底部横向排列）：
  1. **编辑**：[✏️ 编辑]

     - [取消]：次要按钮
     - [保存]：主按钮绿色主题

## 交互说明

### 查看检查点
- 检查点列表按顺序显示所有检查点
- 每个检查点卡片展示完整信息

### 编辑检查点
1. 点击检查点的"编辑"按钮
2. 打开编辑对话框，填充当前检查点数据
3. 修改表单内容
4. 点击"保存"更新检查点
5. 点击"取消"放弃修改

### 删除检查点
1. 点击检查点的"删除"按钮
2. 显示确认对话框："确定要删除这个检查点吗？"
3. 确认后删除检查点
4. 取消后保持不变

### 新增检查点
1. 点击底部的"+ 添加检查点"按钮
2. 打开新增对话框（空表单）
3. 填写表单内容
4. 点击"保存"添加到列表末尾
5. 点击"取消"放弃新增

### 调整顺序
- 点击"上移"按钮：检查点向上移动一位
- 点击"下移"按钮：检查点向下移动一位
- 第一个检查点的"上移"按钮禁用
- 最后一个检查点的"下移"按钮禁用

### 确认并开始教学
1. 点击"确认并开始教学"按钮
2. 验证检查点列表（至少有1个检查点）
3. 验证通过后：
   - 保存检查点计划
   - 跳转到教师模式主界面 `/teacher/view`
4. 验证失败时显示错误提示

### 返回
- 点击导航栏的"返回"按钮
- 确认对话框："检查点计划未保存，确定要返回吗？"
- 确认后返回教师模式配置页
- 取消后停留在当前页

## API 接口

```typescript
// 获取检查点计划
GET /checkpoint-plans/{plan_id}

Response:
{
  "plan_id": 1,
  "topic": "...",
  "checkpoints": [...]
}

// 更新检查点计划
PUT /checkpoint-plans/{plan_id}

Request:
{
  "checkpoints": [
    {
      "title": "Python函数简介",
      "key_point": "函数的概念、作用",
      "question": "函数是什么？"
    }
  ]
}

Response:
{
  "success": true
}

// 开始教学
POST /teacher/sessions/{session_id}/start

Response:
{
  "success": true,
  "redirect_to": "/teacher/view?session_id=1"
}
```

## 设计备注

- 检查点卡片采用黄色便签风格，便于编辑
- 每个检查点的操作按钮清晰可见
- 编辑对话框采用模态框形式，聚焦用户注意力
- 上移/下移按钮直观易懂
- 删除操作需要确认，防止误删
- 新增检查点添加到列表末尾，可通过上移调整位置
- "确认并开始教学"按钮突出显示，引导用户完成配置
- 返回时提供确认提示，防止数据丢失
- 整体保持绿色主题（教师模式）
