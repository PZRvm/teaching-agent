## 页面描述

用户进入系统的第一个页面，用于选择教学模式（观察模式 / 教师模式）。页面采用居中布局，突出两个主要模式选项，底部简要说明技术栈。

## 组件列表

### 1. 导航栏
- **位置**：页面顶部固定
- **内容**：
  - 左侧：系统标题"教学智能体"（手绘大字）
  - 右侧：两个按钮 - "教学历史"、"GitHub"
- **样式**：
  - 背景：白色半透明 `rgba(255, 255, 255, 0.95)`
  - 底部边框：2px 手绘虚线
  - 按钮使用次要按钮样式

### 2. 主标题区
- **内容**：
  - 主标题："教学智能体"（48px 粗体）
  - 副标题："多Agent教学模拟系统"（24px 中等）
- **样式**：
  - 居中对齐
  - 标题下方添加手绘下划线装饰
  - 使用马克笔黑色

### 3. 模式选择卡片区
- **布局**：两列等宽布局，间距 48px
- **卡片尺寸**：每个卡片 400px × 320px

#### 观察模式卡片
- **背景**：蓝色便签色 `var(--sticky-blue)`
- **边框**：3px 马克笔蓝色 `var(--marker-blue)`
- **内容**：
  - 图标：👁️ (48px)
  - 标题："观察模式"（32px 粗体）
  - 描述："观看AI教师与学生自动互动，用于教育研究和教学模式分析"（16px）
  - 按钮："开始观察 →"（主按钮样式，蓝色主题）
- **装饰**：
  - 左上角胶带效果
  - 卡片右下角添加小手绘箭头指向按钮

#### 教师模式卡片
- **背景**：绿色便签色 `var(--sticky-green)`
- **边框**：3px 马克笔绿色 `var(--marker-green)`
- **内容**：
  - 图标：👨‍🏫 (48px)
  - 标题："教师模式"（32px 粗体）
  - 描述："扮演教师角色，控制AI学生进行教学互动，体验教学流程"（16px）
  - 按钮："开始教学 →"（主按钮样式，绿色主题）
- **装饰**：
  - 右上角胶带效果
  - 卡片右上角添加手绘星星装饰

### 4. 底部说明区
- **内容**："技术栈：FastAPI + React + Qwen + SQLite"
- **样式**：
  - 居中对齐
  - 小字 14px
  - 灰色文字
  - 上方添加手绘分割线

## 交互说明

1. **观察模式卡片**
   - 鼠标悬停：卡片轻微上移（translateY -4px），阴影加深
   - 点击"开始观察"：跳转到观察模式配置页 `/observation/config`

2. **教师模式卡片**
   - 鼠标悬停：卡片轻微上移（translateY -4px），阴影加深
   - 点击"开始教学"：跳转到教师模式配置页 `/teacher/config`

3. **导航栏按钮**
   - "教学历史"：跳转到教学历史页 `/history`
   - "GitHub"：打开 GitHub 仓库链接

## 设计备注

- 页面整体采用居中对称布局，突出两个主要选项
- 卡片使用不同颜色区分模式（蓝色=观察，绿色=教师）
- 保持简洁，避免过多装饰干扰核心选择
- 响应式：在小于 1024px 时卡片改为垂直排列
