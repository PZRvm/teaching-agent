# 答辩数据分析页面设计

## 概述

为毕业答辩设计的教学智能体效果分析前端页面。通过多种 ECharts 图表直观展示三种教学模式（灌输式、启发式、讨论式）的教学效果差异。

## 技术选型

- **图表库**：ECharts（`echarts` + `echarts-for-react`）
- **样式**：styled-components（与项目一致）
- **数据**：静态模拟数据，硬编码在组件中
- **风格**：学术正式风格（白底、蓝色系主色调、整洁排版）

## 页面结构

### 路由

新增路由 `/analytics`，对应 `AnalyticsDashboard` 页面组件。

### 布局

顶部标题栏 + Tab 切换栏 + 图表内容区。

```
┌──────────────────────────────────────────────────┐
│  📊 教学智能体效果分析                              │
├──────────────────────────────────────────────────┤
│  [总览]  [学习成效]  [学生参与度]  [学生满意度]     │
├──────────────────────────────────────────────────┤
│                                                  │
│  （当前 Tab 的图表内容）                            │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 统一配色

三种教学模式对应三种颜色，全局一致：
- **灌输式**：`#3B82F6`（蓝色）
- **启发式**：`#A855F7`（紫色）
- **讨论式**：`#22C55E`（绿色）

## 各 Tab 设计

### Tab 1：总览

全局视角，一眼看出三种模式的核心差异。

**顶部：三个指标卡片**（一行三列）

| 卡片 | 内容 |
|------|------|
| 灌输式综合评分 | 72.5 / 100 |
| 启发式综合评分 | 85.3 / 100 |
| 讨论式综合评分 | 91.8 / 100 |

综合评分 = 学习成效×30% + 参与度×30% + 满意度×40%

**中部左：柱状图** — 四维度均分对比

- 类型：分组柱状图（grouped bar）
- X 轴：学习成效、学生参与度、教学效率、学生满意度
- Y 轴：分数（0-100）
- 三组柱子分别对应三种模式

**中部右：雷达图** — 综合能力画像

- 类型：雷达图（radar）
- 五个维度：知识掌握、思维能力、参与度、满意度、课堂氛围
- 三条线分别对应三种模式

**底部：折线图** — 各检查点阶段效果趋势

- 类型：折线图（line）
- X 轴：检查点 1 ~ 5
- Y 轴：综合得分
- 三条折线分别对应三种模式

### Tab 2：学习成效

**左上：柱状图** — 各检查点正确率对比

- 类型：分组柱状图
- X 轴：检查点 1 ~ 5
- Y 轴：正确率（%）
- 三组柱子对应三种模式

**右上：饼图** — 作业评级分布

- 类型：三个饼图并排（ring / doughnut）
- 每个饼图对应一种模式
- 分区：优秀(A)、良好(B)、及格(C)、不及格(D)
- 中间显示该模式的平均评级

**底部：折线图** — 知识掌握度变化趋势

- 类型：面积折线图（area line）
- X 轴：教学时间节点（检查点 1 ~ 5）
- Y 轴：知识掌握度（0-100）
- 三条面积线对应三种模式

### Tab 3：学生参与度

**左上：柱状图** — 主动提问次数 & 回答次数

- 类型：分组柱状图
- X 轴：三种模式
- Y 轴：次数
- 每个模式两根柱子：提问次数（实心） + 回答次数（条纹）

**右上：饼图** — 学生参与类型分布

- 类型：三个饼图并排（ring / doughnut）
- 每个饼图对应一种模式
- 分区：积极发言、被动回答、旁听学习
- 中间显示该模式的活跃学生比例

**底部：折线图** — 各检查点参与度变化

- 类型：折线图（line）
- X 轴：检查点 1 ~ 5
- Y 轴：参与度（0-100）
- 三条折线对应三种模式

### Tab 4：学生满意度

**左上：饼图** — 满意度等级分布

- 类型：三个饼图并排（ring / doughnut）
- 每个饼图对应一种模式
- 分区：非常满意、满意、一般、不满意
- 中间显示该模式的平均满意度

**右上：雷达图** — 满意度细分维度

- 类型：雷达图（radar）
- 五个维度：内容质量、互动体验、难度适中性、节奏感、收获感
- 三条线对应三种模式

**底部：柱状图** — NPS 净推荐值对比

- 类型：普通柱状图
- X 轴：三种模式
- Y 轴：NPS 得分（-100 ~ 100）
- 三根柱子，颜色区分

## 模拟数据设计

数据需体现合理的教学效果差异：

- **灌输式**：学习成效中等，参与度低，满意度一般
- **启发式**：学习成效较好，参与度中等，满意度较好
- **讨论式**：学习成效最好，参与度最高，满意度最高

所有数值模拟 30 名学生一轮完整教学的统计数据。
